- Purpose: `cpm` is a Rust-native Copilot asset manager with a Rust core, clap CLI, and Python wrapper/bindings.
- Tech stack: Rust workspace (`cpm-types`, `cpm-core`, `cpm-cli`), Python package under `python/cpm`, pytest integration tests.
- Current implemented asset kinds in code: plugin, skill, agent, mcp. Architecture doc also describes hooks/workflows, but those are not yet fully wired through types/core.
- Key files: `crates/cpm-cli/src/commands/*` for CLI, `crates/cpm-core/src/project.rs` for manifest/lock/install orchestration, `crates/cpm-core/src/installer.rs` for install paths, `crates/cpm-types/src/lib.rs` for shared models.
